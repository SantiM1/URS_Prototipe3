Cartografía de Hogares de la Unidad del Registro Social

Servicio WMS

El servicio WMS (Web Map Service) permite visualizar mapas desde un servidor geoespacial.

Servicio WFS

El servicio WFS (Web Feature Service) permite obtener datos geoespaciales en formato vectorial.

Servicio WMTS

El servicio WMTS (Web Map Tile Service) proporciona mapas en teselas para una carga más rápida.

KML

Descarga los datos de Cartografía de Hogares en formato KML, compatible con herramientas como Google Earth.
